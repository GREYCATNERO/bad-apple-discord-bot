# 🎵 Bad Apple — бот для Discord

Анимированная ASCII‑версия клипа *Bad Apple* прямо в вашем чате.  
Основано на проекте [NPCat/bad-apple-bot](https://github.com/NPCat/bad-apple-bot) — все права на исходную идею принадлежат автору.

---

## 🧰 Что нужно установить

Установите **Python 3.8 или новее** 

https://www.python.org/

затем выполните в командной строке:

bash
```pip install discord.py Pillow opencv-python```

Модуль time уже есть в Python, его устанавливать не нужно.


------------------------------------------------------------------------------------
## 🚀 Как запустить бота

1)  Скачайте файлы из этой папки (или клонируйте репозиторий).
    Убедитесь, что в папке лежат:

        bad apple discord player.py — основной скрипт

        bad_apple.mp4 — видеофайл (можно взять любой, но имя должно совпадать)

2)  Создайте бота в Discord:
    - Зайдите на https://discord.com/developers/applications
    - Нажмите «New Application», дайте имя.
    - В левом меню выберите Bot → «Add Bot».
    - Скопируйте токен (кнопка «Copy» под названием бота).

3) Включите нужные права:
    - В том же разделе Bot прокрутите вниз до Privileged Gateway Intents.
    - Включите Message Content Intent (это обязательно).

4) Вставьте токен в код:
    - Откройте bad apple discord player.py в любом текстовом редакторе.
    - Найдите в самом низу строку:

```TOKEN = "ВАШ_ТОКЕН_БОТА"```

    - Замените "ВАШ_ТОКЕН_БОТА" на скопированный токен (кавычки оставьте).

5) Запустите бота:
    - Откройте командную строку в папке со скриптом.
    - Выполните:
    ```python "bad apple discord player.py"```
    - При первом запуске бот сам создаст папку frames и сгенерирует все кадры из видео (это займёт 2–3 минуты).
    - Дождитесь сообщения в консоли: Загружено ... кадров.

6) Пригласите бота на свой сервер:
    - На той же странице разработчика перейдите в OAuth2 → URL Generator.
    - Выберите bot и applications.commands.
    - В разделе Bot Permissions поставьте Administrator (или хотя бы Send Messages, Read Messages).
    - Скопируйте сгенерированную ссылку, откройте её в браузере и выберите свой сервер.

------------------------------------------------------------------------------------
## 🎮 Как пользоваться в Discord
В любом текстовом канале напишите команду:
```/badapple```

Бот начнёт отправлять кадры один за другим.
Когда воспроизведение закончится, он напишет «✅ Воспроизведение завершено!».

> ⚠️ Если бот не отвечает на /badapple – подождите 10–20 секунд после запуска (он синхронизирует команды с Discord).

-------------------------------------------------------------------------------------
##🔧 Возможные проблемы

    - Ошибка «Cannot send an empty message» — скорее всего, папка frames пуста или нет видео. Удалите папку frames и перезапустите бота — он создаст её заново.

    - Бот не видит команду — убедитесь, что в настройках бота включён Message Content Intent и бот имеет право читать сообщения на сервере.

    - Слишком медленно / слишком быстро — отрегулируйте переменную TIMEOUT в коде (в начале файла). Меньше число = быстрее.
